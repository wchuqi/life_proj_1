
# 题目

## 162.寻找峰值
https://leetcode.cn/problems/find-peak-element/description/?envType=problem-list-v2&envId=6uaxYMyj

```java
class Solution {
    public int findPeakElement(int[] nums) {
        int res =0;
        for (int i=1;i<nums.length;i++) {
            if (nums[i] > nums[res]) {
                res = i;
            }
        }
        return res;
    }
}
```
