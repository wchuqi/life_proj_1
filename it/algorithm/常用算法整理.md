# 思想

算法思维最重要
面对算法问题，要有一个合理的思考路径
培养思考问题的方式
在学习和实践做题之间，要掌握平衡
空间和时间的交换：哈希表

解决算法问题的整体思路：
- 1、注意题目中的条件（暗示要使用的解法）
- 2、没有思路时：对测试用例，试验一下；不要忽视暴力解法，暴力解法通常是思考的起点
- 3、边界条件的判断
